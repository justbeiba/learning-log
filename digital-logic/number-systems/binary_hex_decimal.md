### 1. Binary
Random decimal - 197

Binary = 11000101

Decimal = 1 + 4 + 64 + 128 = 197
#
### 2. Hex (Hexadecimal)
Hex = C5

A = 10, B = 11, C = 12, D = 13, E = 14, F = 15

C5 is 12 and 5, just map the numbers and group them:
```
12 is 1100
5 is 0101
Group them:
Binary = 1100 0101
```
#
### 3. Octal

Octal = 305

Decimal = 3 * 8^2 + 5 * 8^0 = 192 + 5 = 197

If you want to convert octal to binary, there is a 3-bit direct mapping.
